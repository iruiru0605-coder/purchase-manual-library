# シート構成

このツールは、スプレッドシートを「正」として使います。GoogleドライブはPDFや写真を置く倉庫で、探すときは必ず `商品一覧` から入ります。

## `商品一覧`

正式に管理する商品を置くメインのタブです。

| 列 | 列名 | 型 | 内容 |
|---:|---|---|---|
| A | 商品ID | 固定テキスト | Apps Scriptが `P0001` のように自動採番します。数式や手入力は使いません。 |
| B | ステータス | プルダウン | `使用中`, `処分済`, `売却済` |
| C | 購入日 | 日付 | 必須。保証期限計算に使います。 |
| D | メーカー | テキスト | 必須。検索や公式マニュアル探しに使います。 |
| E | 商品名 | テキスト | 必須。人間が探すときの中心項目です。 |
| F | 型番 | テキスト | 修理依頼、買い替え、公式マニュアル検索で使います。 |
| G | カテゴリ | プルダウン | 紙ファイル置き場の自動採番にも使います。 |
| H | 購入元 | プルダウン | Amazon、楽天、実店舗など。 |
| I | 価格 | 数値 | 任意。修理か買い替えかの判断材料になります。 |
| J | メーカー保証(月) | 数値 | 保証期限計算に使います。 |
| K | 延長保証(月) | 数値 | 任意。店舗の延長保証などを月数で入れます。 |
| L | 保証期限 | 数式 | メーカー保証と延長保証の長い方で自動計算します。 |
| M | 保証状態 | 数式 | `保証中`, `あとN日`, `期限切れ` を表示します。 |
| N | フォルダリンク | URL | Apps Scriptが商品ごとのGoogleドライブフォルダを作って書き込みます。 |
| O | 設置場所 | テキスト | 任意。例: `キッチン`, `書斎` |
| P | メモ | テキスト | 任意。シリアル番号、注意点、紙箱メモなど。 |
| Q | マニュアル状態 | プルダウン | `未確認`, `公式PDFあり`, `公式なし`, `紙あり`, `後でスキャン`, `不要` |
| R | 紙ファイル置き場 | テキスト | カテゴリから自動作成します。あとから編集できます。 |
| S | 公式マニュアルURL | URL | 任意。見つかった公式PDFやサポートページを入れます。 |
| T | スキャン状態 | プルダウン | `未スキャン`, `スキャン済`, `不要` |

推奨数式:

```text
L2:
=IF(OR(C2="",J2=""),"",EDATE(C2,MAX(J2,N(K2)))-1)

M2:
=IF(L2="","",IF(B2<>"使用中","─",IF(TODAY()>L2,"期限切れ",IF(L2-TODAY()<=30,"あと"&(L2-TODAY())&"日","保証中"))))
```

L列とM列は、使いそうな行数まで下へコピーしておきます。スクリプトで登録した行には、自動でも数式が入ります。

## `まとめ登録`

Amazonや楽天などで買った商品が数件たまったときに使う、一括登録用のタブです。

| 列 | 列名 | 内容 |
|---:|---|---|
| A | 取込状態 | 空欄または `未取込` の行を取り込みます。`登録済` と `スキップ` は無視します。 |
| B | 購入日 | 必須 |
| C | メーカー | 必須 |
| D | 商品名 | 必須 |
| E | 型番 | 任意 |
| F | カテゴリ | 任意。紙ファイル置き場の採番に使います。 |
| G | 購入元 | 任意 |
| H | 価格 | 任意 |
| I | メーカー保証(月) | 任意 |
| J | 延長保証(月) | 任意 |
| K | 設置場所 | 任意 |
| L | マニュアル状態 | 任意。空欄なら `未確認` になります。 |
| M | 紙ファイル置き場 | 任意。空欄ならカテゴリから自動作成します。 |
| N | 公式マニュアルURL | 任意 |
| O | スキャン状態 | 任意 |
| P | メモ | 任意 |
| Q | 生成ID | 取り込み後にスクリプトが書き込みます。 |
| R | フォルダリンク | 取り込み後にスクリプトが書き込みます。 |

最低限、`購入日`、`メーカー`、`商品名` の3つがあれば登録できます。

## `設定`

プルダウン候補や紙分類コードを置くタブです。

| 列 | 列名 | 内容 |
|---:|---|---|
| A | ステータス | プルダウン候補 |
| B | カテゴリ | プルダウン候補 |
| C | 購入元 | プルダウン候補 |
| D | 紙分類コード | 紙ファイル置き場に使うカテゴリコード |
| E | 初期設定 | 設定名 |
| F | 値 | 設定値 |
| G | メモ | 設定メモ |
| H | マニュアル状態 | プルダウン候補 |
| I | スキャン状態 | プルダウン候補 |
| J | 取込状態 | プルダウン候補 |

紙分類コードの例:

| カテゴリ | 紙分類コード |
|---|---|
| キッチン家電 | KI |
| 生活家電 | SE |
| PC・周辺機器 | PC |
| スマホ・充電 | SM |
| AV機器 | AV |
| 家具・設備 | FU |
| 工具 | TO |
| その他 | OT |

例えばカテゴリが `キッチン家電`、紙分類コードが `KI` の場合、紙ファイル置き場は `キッチン家電（KI）の001` のように作られます。
