# Sheet Schema

## `商品一覧`

This tab is the source of truth.

| Col | Name | Type | Notes |
|---:|---|---|---|
| A | 商品ID | Fixed text | Generated by Apps Script, e.g. `P0001`. Do not use formulas. |
| B | ステータス | Dropdown | `使用中`, `処分済`, `売却済` |
| C | 購入日 | Date | Required |
| D | メーカー | Text | Required |
| E | 商品名 | Text | Required |
| F | 型番 | Text | Useful for repairs and manual search |
| G | カテゴリ | Dropdown | Used for paper-storage numbering |
| H | 購入元 | Dropdown | Amazon, Rakuten, store, etc. |
| I | 価格 | Number | Optional |
| J | メーカー保証(月) | Number | Required for warranty calculation |
| K | 延長保証(月) | Number | Optional |
| L | 保証期限 | Formula | Uses the longer of J/K |
| M | 保証状態 | Formula | `保証中`, `あとN日`, `期限切れ` |
| N | フォルダリンク | URL | Generated by Apps Script |
| O | 設置場所 | Text | Optional |
| P | メモ | Text | Serial number, paper box note, etc. |
| Q | マニュアル状態 | Dropdown | `未確認`, `公式PDFあり`, `公式なし`, `紙あり`, `後でスキャン`, `不要` |
| R | 紙ファイル置き場 | Text | Generated from category, editable afterward |
| S | 公式マニュアルURL | URL | Optional |
| T | スキャン状態 | Dropdown | `未スキャン`, `スキャン済`, `不要` |

Recommended formulas:

```text
L2:
=IF(OR(C2="",J2=""),"",EDATE(C2,MAX(J2,N(K2)))-1)

M2:
=IF(L2="","",IF(B2<>"使用中","─",IF(TODAY()>L2,"期限切れ",IF(L2-TODAY()<=30,"あと"&(L2-TODAY())&"日","保証中"))))
```

Fill these formulas down for the expected working range.

## `まとめ登録`

Use this tab when several purchases have accumulated.

| Col | Name | Notes |
|---:|---|---|
| A | 取込状態 | Empty or `未取込` rows are imported. `登録済` and `スキップ` are ignored. |
| B | 購入日 | Required |
| C | メーカー | Required |
| D | 商品名 | Required |
| E | 型番 | Optional |
| F | カテゴリ | Optional, used for paper-storage numbering |
| G | 購入元 | Optional |
| H | 価格 | Optional |
| I | メーカー保証(月) | Optional |
| J | 延長保証(月) | Optional |
| K | 設置場所 | Optional |
| L | マニュアル状態 | Optional, defaults to `未確認` |
| M | 紙ファイル置き場 | Optional. If blank, generated from category. |
| N | 公式マニュアルURL | Optional |
| O | スキャン状態 | Optional |
| P | メモ | Optional |
| Q | 生成ID | Written after import |
| R | フォルダリンク | Written after import |

## `設定`

Recommended columns:

| Col | Name | Purpose |
|---:|---|---|
| A | ステータス | Dropdown source |
| B | カテゴリ | Dropdown source |
| C | 購入元 | Dropdown source |
| D | 紙分類コード | Category code used in paper-storage labels |
| E | 初期設定 | Setting name |
| F | 値 | Setting value |
| G | メモ | Setting note |
| H | マニュアル状態 | Dropdown source |
| I | スキャン状態 | Dropdown source |
| J | 取込状態 | Dropdown source |

| 生活家電 | SE |
| PC・周辺機器 | PC |
| スマホ・充電 | SM |

| 家具・設備 | FU |
