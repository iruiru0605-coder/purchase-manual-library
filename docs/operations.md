# 運用手順

このページは、日常的に取説ライブラリを使う時の手順です。

## 基本の流れ

```text
購入履歴を取り込む
↓
候補を仕分ける
↓
AI補完する
↓
PDF候補を探す
↓
登録する
↓
ライブラリで検索・編集する
```

## 1. 購入履歴を取り込む

### CSVがある場合

1. `購入履歴Inbox` を開く
2. 購入元を選ぶ
3. CSVを選択する
4. `取り込む` を押す

列名が完全一致していなくても、商品名、購入日、価格、注文番号などは可能な範囲で自動判定します。

### CSVがない場合

Amazonの公式データリクエストは時間がかかる場合があります。普段は購入履歴ページから直接コピーする方法が早いです。

1. Amazonまたは楽天の購入履歴ページを開く
2. 注文カード周辺を範囲選択してコピーする
3. アプリの `購入履歴ページを貼り付け` に貼る
4. `貼り付けから取り込む` を押す

楽天は、`注文日` と `注文番号` が別行になる形式にも対応しています。

## 2. 登録候補を仕分ける

候補ごとに次を選びます。

* `登録する`: ライブラリに入れたい商品
* `不要`: 食品、日用品、消耗品など
* `後で確認`: 判断を保留したい商品
* `削除`: 誤って取り込んだ候補

家電、家具、工具、子ども用品、健康器具、車用品、ソフトウェアなどは登録対象になりやすいです。

## 3. AI補完

`AI補完` を押すと、次を補完します。

* メーカー
* 商品名
* 型番
* カテゴリ
* PDF探索用の検索キーワード

LLM APIキー未設定時は、CSV列と商品名から簡易推定します。

## 4. PDF候補探索

`候補を探す` を押すと、公式サイト優先でPDFを探します。

PDF候補は自動選択されます。説明書ではないPDFが混ざっている場合は、登録前にチェックを外してください。

## 5. 登録する

`選択したPDFを保存して登録` を押します。

Google Driveにログイン済みならDriveへ保存します。未ログインならローカルの `.manual-library/archive/` へ保存します。

## 6. ライブラリで探す

`ライブラリ` タブで検索できます。

検索対象:

* 商品名
* メーカー
* 型番
* カテゴリ
* 紙マニュアル保管先

カテゴリはプルダウンでも絞り込めます。

## 7. 登録後に修正する

ライブラリの商品カードで、次を後から変更できます。

* メーカー
* 商品名
* 型番・品番
* カテゴリ
* 購入日
* メーカー保証月数
* 延長保証月数
* 紙マニュアル保管先
* メモ

変更したら `保存` を押してください。

## 8. PDFを後から追加する

公式PDFが見つからない場合や、紙マニュアルをスキャンした場合の手順です。

1. `ライブラリ` を開く
2. 該当商品を探す
3. `PDF追加・入れ替え` で種類を選ぶ
4. PDF表示名を入力する
5. `PDFを選択` でファイルを選ぶ
6. `PDFを追加` を押す

PDFを入れ替えたい場合は、古いPDFを削除してから新しいPDFを追加します。

## 9. 誤って登録した商品を削除する

ライブラリの商品カードで `商品を削除` を押します。

ローカル保存PDFがある場合は、商品削除時にローカルPDFファイルも削除します。Google Drive上に保存済みのファイルは、必要に応じてDrive側でも確認してください。

## 10. 紙マニュアル

登録時に次の形式で保管先を自動作成します。

```text
カテゴリ BOX-番号 #連番
```

例:

```text
キッチン家電 BOX-01 #001
工具 BOX-06 #003
```

紙の整理ルールを変えたい場合は、ライブラリの商品カードで `紙マニュアル保管先` を直接編集できます。
